### How to start the app

- Client
  ```
  $ cd client
  $ docker build -t ocard-client  .
  $ docker run -it -p 3000:3000 ocard-client
  ```

- Server
  ```
  $ cd server
  $ docker build -t ocard-server  .
  $ docker run -p 7777:80 ocard-server 
  ```
